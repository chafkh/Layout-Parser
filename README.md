# Webscrapping project with Deep Learning

## 📌 Project Description

This is a project I carried out during my intership in a research laboratory. During this internship, I assisted an economics researcher with a research project, specifically with collecting data for analysis.

I was asked to webscrape data from scanned PDF files using Deep Learning.

---

## 📊 Project Preview

- Python Notebook: [Projet LayoutParser.ipynb](./Projet%20LayoutParser.ipynb)

## ⚙️ Main steps
This project follows a series of logical steps:
- Importing the necessary libraries
- Conversion to jpg format
- Model implementation
- save into excel file for analysis
